# Cosmic Connect

An AI-powered astrology platform connecting users with the cosmos.

## Features
-   **AI Astrology Chat**: Ask questions about your career, love life, and health (Simulated & Scientific).
-   **Live Consultations**: Connect with expert astrologers (Simulated).
-   **Daily Predictions**: Get personalized daily horoscopes.
-   **Kundali Generation**: Generate your Vedic birth chart.

## Tech Stack
-   React + Vite
-   TypeScript
-   TailwindCSS + Shadcn/UI
-   Supabase (Backend/Database)

## Setup for Development
1.  Clone the repo.
2.  Run `npm install`.
3.  Run `npm run dev`.

## Deployment
This project is configured for **Vercel**.
-   **SPA Routing**: `vercel.json` included to handle client-side routing.
-   **Environment Variables**: Add `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` in Vercel settings.
